Project Assignment A: Requirements and Design

Project Overview

The website being created is a site for a local educator. It's purpose is to provide current students, prospective students, as well as their families to learn more about the way things are tought throughout the department. The main purpose of this site is to make it easier for students to have access to the educator, as well as things such as her schedule, meeting times, office hours, or even post questions that will be answered in a question board.

Client Information

  • Krista Chapman
  • Behavior Liaison Specialist for Kannapolis City Schools' Exceptional Children Department
  • krista.chapman@kcs.k12.nc.us
  • 704-273-2437

Pages

Home

  • The main page for the site
  • Users will use this page as a basis to bring introduce them to the service
  • Content will consist of picture, a brief description, as well as the main links to other pages
  • No data will be needed in this page
  • The buttons will consist of the main navigation for the entire site
  • There will be a scrollable image cycle to view several pictures

Student Spotlight

  • This page will feature several willing students, as well as many things that the deparment will provide
  • Users will be able to view this page, and admins can also edit the spotlights as well
  • This will consist of pictures, as well as descriptions of students or services that the school will bring
  • A user can click on a picture, and it will expand a longer, more detailed description of that photo

Calendar

  • This page will feature an up-to-date with Krista's calendar
  • Users will be able to view a calendar, as well as make appointments with Krista, when available
  • Users will have to link their google calendar in order to make an appointment
  • After an appointment is made, both google calendars will be updated

About

  • This will be an information-only page
  • Users will be able to view this page
  • Page will consist of the mission statement, as well as important information for the school and department

Contact

  • Contact and message board page
  • Users will be able to view this page, and admin will also use it as well
  • The content of this page will be a content form, as well as a question and answer board
  • User will enter their data such as email and name, as well as a question to ask Krista directly
  • The data fields that will need validations is email, name, and subject matter
  • Users will be able to view the questions that other users have asked, and also the answers to those questions

Dynamic Functionality

  • Javascript is used for this website throughout. The first you will see this is the homepage. It will consist of a scrollable image wheel that a user can click on arrows to scroll through the images.
  • In the calendar page, javscript is used with the calendar, and will use Google's Calendar API functionality
  • In the contact page, the main functinality is the contact form and message board. Users will be able to ask questions, and recieve answers from an admin account. Javascript will keep these messages up-to-date, and will also keep any login information from a user's account. It will also deferentiate the user accounts from the admin accounts in the message board.
